[Appendix] Output format

In addition to arbitrary characters, the following several built-in variables and escape sequences are available in the Output format.

Built-in variables

The following built-in variables are available for Output setting. If $${FOO} is written, the built-in variables are not expanded, but are expanded to the string ${FOO}.
${url}
Current URL
${url_simple}
Current URL containing only origin and pathname (i.e. URL without search and hash)
${file_name}
File name
${boxdrive_mac}
File path used by Box Drive for macOS with File Provider Extension mode. / is used as a path separator. ~/Library/CloudStorage/Box-Box is used as the mount directory.
${boxdrive_mac_old}
File path used by Box Drive for macOS with kernel-based mode. / is used as a path separator. ~/Box is used as the mount directory.
${boxdrive_win}
File path used by Box Drive for Windows. \ is used as a path separator. %USERPROFILE%\Box is used as the mount directory.

Escape sequences

The following escape sequences are available in the Output setting. If the string \\ is used, the escape sequence is not expanded and the string is replaced by the string \.
\n
Replaced by a newline character.
\t
Replaced by a tab character.
\\
Replaced by \.

Badge Settings

Add Row

Column Description

Start
Search start byte position in the file. If omitted, it is assumed that the range is specified from the beginning of the file.
Length
Byte length from the search start position to the search end position. If omitted, it is assumed that the range is specified up to the end of the file.
Search
String to search. You can specify the value in hexadecimal by \xnn. The \ itself is specified as \\.
Sign
Text to put on the badge.
FG Color
Color of the badge sign.
BG Color
Background color of the badge sign.
